# smartBusBookingSystem

A simple C++ application for managing bus bookings.

## Features
- Add, view, and manage bus bookings
- User-friendly console interface
- Fast and lightweight

## Usage
1. Build or run the provided `main.exe`.
2. Follow on-screen instructions to interact with the system.

## Requirements
- Windows OS
- C++ compiler (for building from source)
